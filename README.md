# Personality Predictor

An interactive **Personality Predictor** that asks **15 thought-provoking questions** to determine the user's personality type based on **Introvert/Extrovert, Thinking/Feeling, Judging/Perceiving** traits.  

It is **fully offline**, responsive, and visually appealing, making it perfect for students, hobby projects, or portfolio showcases.

---

## Features

- 15 curiosity-driven questions to keep users engaged  
- Offline & lightweight – no APIs needed  
- Interactive and modern UI  
- Radio buttons and buttons with hover effects  
- Personality calculation logic for MBTI-like traits  
- Easy to expand – add more questions or traits dynamically  
- Fully offline – works in any modern browser  

---

## Technologies Used

- **HTML5** – Structure of the project  
- **CSS3** – Styling and animations  
- **JavaScript (ES6)** – Dynamic rendering and personality logic  
- **Responsive Design** – Works on desktop, tablet, and mobile  
- **Interactive Elements** – Hover animations, radio buttons, and results display

## How It Works
- Questions are dynamically rendered using JavaScript.

- Each answer contributes points to a specific personality trait (Introvert, Extrovert, Thinking, Feeling, Judging, Perceiving).

- After answering all questions, the script calculates the dominant traits.

- The result is displayed as a personality type, e.g., Introvert-Thinking-Judging.
